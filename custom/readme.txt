Content of this directory is not required!
You can add/edit/delete content as you want)

For example, you can add here file with class, which has the same name as core system class,
in this case your class will be used (may be useful in certain cases when modification of system files is needed).
Also class can be placed not in "cs" namespace, but in "cs\custom" - then custom class can even inherit system one, and redefine default behavior.

WARNING: This directory may contain files from custom build of CleverStyle Framework or files created by components, so edit content carefully.
